Создал python-скрипты для создания sql-скрипта импорта данных из базы ФИАС из формата DBF.
Они используют либу dbfread.(ещё typing используют, но это можно и выпилить)
Один скрипт для импорта таблицы адрессных объектов (ADDROBJ) - адресов вплоть до улиц.
Второй для импорта таблицы домов (HOUSE).
По хорошему их доделывать надо в один нормальный.
Плюс sql'ки для создания структуры базы.

Саму базу надо ручками выкачать с http://fias.nalog.ru/Updates.aspx там в доке есть ссылка на актуальную например http://fias.nalog.ru/Public/Downloads/Actual/fias_dbf.rar
(можно wget'ом c докачкой)
И распаковать + в python-скрипте путь этот указать.
Создаём таблицы скриптами fias_init*
Натравляем fias_import.py на каталог с базой распакованной, записываем вывод в sql'ку. Выполняем её
на нужной базе.

Ещё нашёл запрос для рекурсивного получения полного адреса - но он только по одному элементу - для полнотекстового надо что-то ещё делать.
Для домов только начал - делеко не продвинулся.

По самой структуре базы на хабре есть статьи, например https://habrahabr.ru/post/316314/
